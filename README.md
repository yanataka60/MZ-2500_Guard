# SHARP MZ-2500 カンガルーポケットガード

![Guard4](https://github.com/yanataka60/MZ-2500_Guard/blob/main/JPEG/Guard4.JPG)

　MZ-2500のカンガルーポケットは壊れやすく、また移動時などに下から手で支えようとしたときに触って壊してしまいそうでひやひやします。

　そこで手が触れないようガードを作ってみました。

　STLフォルダあるmz-2500_Guard.stlを3Dプリンタで出力してください。

![Guard1](https://github.com/yanataka60/MZ-2500_Guard/blob/main/JPEG/Guard1.JPG)

　両面テープでカンガルーポケットの下に貼り付けます。
